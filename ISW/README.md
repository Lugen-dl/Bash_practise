# **Infrastructure Security Watchdog**
## **Вступление**
**Рабочий проект** Open-source инструмент kottster. Позволяет вам работать с
реляционными таблицами из вашей базы данных. 
## **Возможности**
- Полная интеграция с вашей базой данных
- Создание, удаление, перезапись ваших таблиц
- Сборка собственных дашбордов
- Создание кастомных таблиц с помощью React и JavaScript
## **Требования**
- Ваша дб с проектами
- Подключение к kottster
- Деплой/Подключение к инструменту
