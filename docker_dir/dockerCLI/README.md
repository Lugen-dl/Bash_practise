#**Docker-CLI**
##**Вступление**
**Тренировочный** интерактивный инструмент для использования основных функций Docker:
      Создание образов, контейнеров и создание личных Dockerfile сборок.
##**Возможности**
- Поиск и установка образов через Docker hub
- Загрузка образов на локальную систему
- Запуск контейнера с основными флагами
- Сборка Docker-образов из Dockerfile
##**Требования**
- Linux
- Bash
- Docker
- Docker Hub account для аутентификации
##**Запуск**
./main.sh (скрипт автоматически делает файл исполняемым)
##**Установка и запуск**
git clone https://github.com/Lugen-dl/Bash_practise.git
cd Bash_practise/docker_dir/dockerCLI
./main.sh
